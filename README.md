# Daily Schedule App

日付ごとに予定やタスクを管理する Android アプリです。  
学習目的で開発しています。

## 機能
- 今日の日付を表示
- 日付の切り替え
- チェックボックスによる完了・未完了管理
- タスクの表示切替

## 次の改善予定
- アプリ再起動後もタスクを保持できるようにする（SharedPreferences）
- 完了タスクの削除機能
- RecyclerView を用いたUI改善

## 技術スタック
- Java
- Android Studio
- Android SDK
- Calendar API
